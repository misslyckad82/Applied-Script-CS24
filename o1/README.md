# script1.sh

Detta script används för att göra en kontroll av Linux-miljöer. 

## Instruktioner för körning
Kommandot top visar realtidsövervakning av systemet, främst CPU, minne och processer.

Tryck på följande tangenter medan top kör: <br>
p sortera efter CPU <br>
M sortera efter minne <br>
T sortera efter körtid <br>
k döda process (ange PID) För att avbryta kill-prompten tryck crtl+C <br>
1 visa alla CPU-kärnor <br>
h hjälp <br>
c visa hela kommandoraden <br>
u visa processer för viss användare <br>
q avsluta <br>

FÖR ATT AVSLUTA HELA KÖRNINGEN TRYCK CTRL+C
